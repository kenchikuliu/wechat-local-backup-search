# 微信本地备份搜索

这是面向客户交付的本地 Windows 软件。

目标只有两件事：

1. 自动备份自己的微信聊天记录
2. 以后能在本机上快速搜索、定位、查看上下文

整套数据只保存在本地电脑，不上传服务器。

## 现在这版能做什么

- 自动识别 Windows 微信 3.x 数据目录
- 从运行中的微信进程提取数据库密钥
- 增量解密聊天数据库
- 增量导出聊天记录到本地
- 建立本地 SQLite 搜索索引
- 桌面窗口内直接搜索
- 查看命中消息的前后文
- 查看会话摘要
- 查看相关会话和高频关键词
- 支持定时自动备份

## 客户怎么用

1. 先启动微信并保持登录
2. 双击 `WeChatBackupSearch.exe`
3. 第一次点击“立即备份”
4. 等待索引完成
5. 在搜索框输入关键词
6. 点选结果，查看消息详情、前后文、会话摘要
7. 如果要持续备份，勾选“自动备份”并设置间隔分钟数

## 桌面窗口里的主要区域

- 顶部指标
  - 已索引消息数
  - 会话数量
  - 会话摘要数量
  - 当前任务状态
  - 搜索引擎类型
- 中间控制区
  - 立即备份
  - 只重建索引
  - 自动备份
  - 备份间隔
  - 语音转录开关
- 搜索区
  - 关键词
  - 联系人或群名过滤
  - 日期范围过滤
- 结果区
  - 命中消息列表
  - 消息详情
  - 会话摘要
  - 相关会话 / 高频关键词
- 日志区
  - 显示最近的备份和索引日志

## 数据保存位置

默认会自动选一个空间足够的本地磁盘目录，目录名是：

```text
backup_search_data
```

典型内容包括：

```text
backup_search_data/
  decrypted_3x/           # 解密后的数据库
  exported_chats/         # 导出的聊天 JSON
  search_index.sqlite3    # 搜索索引
  settings.json           # 自动备份设置
  backup.log              # 运行日志
```

## 当前这份真实数据的大致体量

这不是软件理论值，是已经测过的一份真实微信数据：

- 总占用约 `5.33 GB`
- 解密数据库约 `4.11 GB`
- 导出聊天 JSON 约 `238.84 MB`
- 搜索索引约 `0.86 GB`

所以这个产品本体很小，但数据目录会随着聊天记录累积持续变大。

## 搜索能力说明

这版不是只做“全文匹配”。

除了关键词搜索，还会额外生成：

- 会话摘要
- 高频关键词
- 发送者分布
- 消息类型分布
- 命中消息前后文
- 相关会话提示

这对于客户后续“快速找到某段聊天、某个合同、某个电话、某次沟通上下文”更有用。

## 安全边界

- 数据默认只保存在本机
- 桌面版不依赖公网服务
- 只有在你主动开启语音转录且配置外部服务时，语音内容才可能出机
- 如果电脑账号本身没有保护，明文聊天记录仍然有泄露风险

